*Frame of reference: Macro policy researcher reviewing scenario outputs for academic use. Primary concerns: methodological consistency, data provenance, reproducibility.*

> Presenter preparation: Know what drove the T4 at step 7 — structural model divergence on Reserve Coverage; it does not affect the constraint boundary established at step 4.

**DEMO-205** · MEDIUM — "Direction stable" assertion without displayed supporting test

> Presenter preparation: Confirm that the direction stability test result is visible in Zone 3 and be ready to navigate there if asked.

**DEMO-206** · MEDIUM — HCL "—" entries with no reason code
> Human Cost Ledger shows dash for several rows without a reason code (Not modelled / Data absent / Suppressed). A researcher documenting provenance gaps cannot tell whether the dash is "we don't model this" or "we model it but have no data."

**DEMO-207** · HIGH — Senegal as ZMB calibration comparator (ECOWAS ≠ SADC)

**DEMO-208** · LOW — Q1 fraction vs poverty headcount population denominator ambiguity
